Adding items to Inventory:

To add items to the database, just use the form below.

Id Numbers:

Id numbers are starting from 1 and increment by 1 everytime a new item is added.

Viewing single items:

To view a certain item, use http://localhost:5000/data/idnumber. For example for item 1, http://localhost:5000/data/1.

Updating single items:

To update a certain item, use http://localhost:5000/data/idnumber/update. For example for item 1, http://localhost:5000/data/1/update.

Deleting single items:

To delete a certain item, use http://localhost:5000/data/idnumber/delete. For example for item 1, http://localhost:5000/data/1/delete.
